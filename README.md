
Face & Mask Detection
========================
2 steps face & mask detection program. The program will identify the face before detecting the mask.


Dependencies
========================
python 3.6

```
pip install -r requirements.txt
```

Executing program
========================
```
python face_mask_detector.py --fps 15 video.mp4 --display
```

fps: frame per second
